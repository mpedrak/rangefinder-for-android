# Rangefinder app for Android OS

## Requirements

- Android version 11 or newer
- Camera with variable focus

## Authors

- Andrzej Wacławik
- Michał Pędrak
- Mikołaj Maślak

## Expected future functions

- Option to focus on specific area on camera preview by clicking it
- Option to use various camera lenses for different ranges and precisions

 ## Possible future functions 

- Option to save measurement with photo to history located on device
- Option for measuring distance between objects
- Option for approximating distances greater than focus range by other technics
